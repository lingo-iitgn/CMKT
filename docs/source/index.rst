Welcome to CMKT's documentation!
================================

Welcome to the official documentation of **CMKT** (Code-Mixed toolKit), a Python library developed to facilitate efficient preprocessing of code-mixed text. This comprehensive toolkit offers an array of functionalities for seamless code-mixed text handling, encompassing data acquisition, text preprocessing (including stemming and tokenization), code-mixed NLP tasks , and a metrics module for quantifying code-mixed text and evaluating model performancs. In this documentation, you will find detailed explanations, examples, and usage instructions for leveraging the features provided by cmkt.

There are four different modules available:-

* Data Acquisition Module
* Preprocessing Module
* Tasks Module
* Metrics Module

Language Convention followed throughout cmkt

* English = "en"
* Hindi = "hi"
* Hindi-English Mixed = "hineng

Check out the :doc:`usage` section for further information, including how to :ref:`install <installation>` the project.
